Step#1 - Run Redis App against Redis Container
1.a Run Redis Containers
    docker run --name redis1 -d -p 6379 redis
1.b Run the sample Redis App
    python redis_driver.py -p 6379

Step#2 - Run Redis App against Enterprise Pack Container
1.a Run Redis Containers
    docker run -d --cap-add sys_resource --name rp -p 8443:8443 -p 12000:12000 redislabs/redis
1.b Run the sample Redis App
    python redis_driver.py -p 12000
